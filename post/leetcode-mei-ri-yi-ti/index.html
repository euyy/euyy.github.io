2032.至少在两个数组中出现的值

给你三个整数数组 nums1、nums2 和 nums3 ，请你构造并返回一个 元素各不相同的 数组，且由 至少 在 两个 数组中出现的所有值组成。数组中的元素可以按 任意 顺序排列。

示例 1：

输入：nums1 = [1,1,3,2], nums2 = [2,3], nums3 = [3]
输出：[3,2]
解释：至少在两个数组中出现的所有值为：
- 3 ，在全部三个数组中都出现过。
- 2 ，在数组 nums1 和 nums2 中出现过。

示例 2：

输入：nums1 = [3,1], nums2 = [2,3], nums3 = [1,2]
输出：[2,3,1]
解释：至少在两个数组中出现的所有值为：
- 2 ，在数组 nums2 和 nums3 中出现过。
- 3 ，在数组 nums1 和 nums2 中出现过。
- 1 ，在数组 nums1 和 nums3 中出现过。

示例 3：

输入：nums1 = [1,2,2], nums2 = [4,3,3], nums3 = [5]
输出：[]
解释：不存在至少在两个数组中出现的值。

提示：

  • 1 <= nums1.length, nums2.length, nums3.length <= 100
  • 1 <= nums1[i], nums2[j], nums3[k] <= 100

思路点拨

方法一：哈希表+位存储

统计不同元素在不同数组中出现的次数即可。

需要注意的是，一个元素可能在一个数组中出现多次。

所以可以使用哈希表来存储一个元素在几个数组中出现过，利用位运算来表示一个元素在第几个数组中出现过。也就是说，把数组 numnumnum 作为哈希表的 keykeykey， 把在哪些数组中出现过作为 valuevaluevalue，如果在第一个数组中出现过，则把第 iii 个二进制位置为 111 。这样就可以通过判断一个数的 valuevaluevalue ，来判断是否在多个数组中出现过。如果 valuevaluevalue 有多余一个二进制 111，则表示有多个数组出现过。

这里，判断一个数中有几个二进制 111 可以使用位运算 num&(num−1)num \& (num - 1)num&(num−1) ，这个运算可以去除掉 numnumnum 中最后一个二进制 111 .

算法实现

class Solution {
public:
    vector<int> twoOutOfThree(vector<int>& nums1, vector<int>& nums2, vector<int>& nums3) {
		unordered_map<int,int> ump; // 定义存储数字以及出现在哪些数组的哈希结构
        for(int n : nums1)
        {
            ump[n] = 1; // 对于第一个数组中的数字，凡是出现过，则把值置为1
        }
        for(int n : nums2)
        {
            ump[n] |= 2; // 对于第二个数组中的数字，凡是出现过，则把第二位二进制置为1
        }
        for(int n : nums3)
        {
            ump[n] |= 4; // 对于第二个数组中的数字，凡是出现过，则把第三位二进制置为1
        }
        vector<int> ret; // 定义存储结果的数组
        for(auto &[k,v] : ump) // 对于哈希表中的每个key-value对，进行判断
        {
            if(v & (v-1)) // 如果v不止有一个二进制1，就把k添加进ret中
            {
                ret.push_back(k);
            }
        }
        return ret;
    }
}
